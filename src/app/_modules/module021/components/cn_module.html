021 ChangeDetectionStrategy.OnPush and Immutability

HTML







Whats going on?
  • Component 1
    ChangeDetectionStrategy.OnPush
    • detects only when whole objects are swapped
    • objects themselves are considered immutable
    • faster
    • Internal property updates inside the component are detected
      and cause an update of the whole object.
      Example:
      Add email property (not detected)
      Internal update (with name change also email appears)
  • Component 2
    ChangeDetectionStrategy.Default
    • watches over every single property of every object
    • much slower
