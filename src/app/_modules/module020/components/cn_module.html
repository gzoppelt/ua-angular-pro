Whats going on?
  • Component 1 encapsulation: ViewEncapsulation.Emulated (default)
    emulates by Angular means how shadow DOM works (not all browsers have shadow DOM yet)
    • does not effect outside
    • uses globals if there is no own property
  • Component 2 encapsulation: ViewEncapsulation.Native
    encapsulates styles by using shadow DOM
    • does not effect outside
    • globals override even if there is an own property
  • Component 3 encapsulation: ViewEncapsulation.None
    no encapsulation of styles
    • effects all components
    • overrides earlier outside properties
    • later outside properties override own properties (cascading)
