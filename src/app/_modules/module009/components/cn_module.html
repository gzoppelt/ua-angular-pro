009 Using ElementRef and nativeElement

HTML

Login

 Whats going on?
  • In AuthFormComponent do:
    • template: input field for email gets a template-ref #email:
      <input type="email" name="email" ngModel #email>
    • styles: class email gets a red border
      .email { border-color: red; }
    • import { ElementRef } from @angular/core;
    • @ViewChild('email') email: ElementRef;
    • ngAfterViewInit:
      this.email.nativeElement.setAttribute('placeholder', 'Enter email address');
      this.email.nativeElement.classList.add('email');
      this.email.nativeElement.focus();
  • All expected changes happen: placeholder appears, border is red, and email has focus.
