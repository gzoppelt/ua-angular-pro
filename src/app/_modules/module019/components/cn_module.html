019 Using ngTemplateOutlet with context

HTML {{ name }} : {{ location }} Whats going on?
  • Template:
    <ng-container
      [ngTemplateOutlet]="tmpl"
      [ngTemplateOutletContext]="ctx"
    ></ng-container>

    <ng-template #tmpl let-name let-location="location">
      {{'{{'}} name }} : {{'{{'}} location }}
    </ng-template>
  • ModuleComponent:
    import { Component } from '@angular/core';

    @Component({
      selector: 'app-module',
      templateUrl: './cn_module.html',
      styleUrls: ['./cn_module.css']
    })
    export class ModuleComponent {
      ctx = {
        $implicit: 'Todd Motto,
        location: 'England, UK'
      };
    }
