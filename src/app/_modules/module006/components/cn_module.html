 Whats going on?
  • New <auth-message> component which has a property days = 7;
    We will amend days to 15 when the component is projected into <auth-form>
  • In AuthFormComponent do:
    • import { ViewChild, AfterViewInit } from @angular/core;
    • import AuthMessageComponent in order to look it up
    • define message:
      @ViewChild(AuthMessageComponent) message: AuthRememberComponent;
    • in ngAfterInit do (forEach comes from the QueryList prototype):
      if (this.remember) {
        this.remember.forEach( item => {
          item.checked.subscribe( checked => this.showMessage = checked );
        });
      }

    • The rest of the module stays unchanged:
  • Projecting the component is straight forward, this is about binding to them.
    <auth-form (submitted)="loginUser($event)">
      <h3>Login</h3>
      <auth-remember (checked)="rememberUser($event)"><auth-remember>
      <button type="submit">Join us</button>
    </auth-form>
  • rememberUser keeps an internal flag updated this.rememberMe = remember;
    which will be submitted together with the login data.
  • AuthRemeberComponent emits the checked event to be used in the AuthFormComponent
    @Component({
      selector: 'auth-remember',
      template: `
        <label>
          <input type="checkbox" (change)="onChange($event.target.checked)"/>
          Keep me logged in
        </label>
      `
    })
    export class AuthRememberComponent {
      @Output() checked = new EventEmitter<boolean>();
      onChange(value: boolean) {
        this.checked.emit(value);
      }
    }
